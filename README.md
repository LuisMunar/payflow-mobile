# payflow-mobile

Android mobile app for the Payflow credit card checkout challenge.

## Stack

- React Native
- TypeScript
- Redux Toolkit
- React Navigation
- Secure/encrypted storage
- Jest
- Android

## Architecture

The app follows a feature-based architecture with Redux/Flux:

- `features/products`
- `features/cart`
- `features/checkout`
- `features/payment`
- `features/transaction-result`
- `shared/api`
- `shared/components`
- `shared/storage`
- `shared/validation`

## Local Setup

Pending implementation. Expected commands:

```bash
npm install
npm run android
```

## Tests

Expected commands:

```bash
npm test
npm run test:cov
```

Coverage target: more than 80%.

## Android APK

The APK can remain in the default Android build output path generated by React Native/Gradle.

## Environment

Copy `.env.example` to `.env` and fill local values.

Do not commit real credentials or private values.

## Git Workflow

Work by feature branches and pull requests.

Recommended first branches:

- `feature/mobile-project-setup`
- `feature/mobile-navigation-redux`
- `feature/mobile-products`
- `feature/mobile-cart-checkout`
